Linux 文件系统快速选型指南

核心对比表

文件系统 最佳使用场景 (Use Case) 安装/配置 (Linux命令) 代表软件/环境
ext4
  • 桌面/笔记本系统盘
  • 通用服务器
  • 需要最大化兼容性的地方
 sudo mkfs.ext4 /dev/sdX 大多数发行版默认 Ubuntu/CentOS 装机默认，所有通用软件
XFS
  • 大文件 (视频、媒体库)
  • 高并发IO (数据库)
  • RHEL/CentOS 默认
 sudo mkfs.xfs /dev/sdX 需安装: xfsprogs
  • PostgreSQL / MySQL
  • 大型视频编辑
  • 邮件服务器
ZFS
  • 海量存储 (NAS)
  • 数据安全 (校验码)
  • 需要快照/容错
 sudo apt install zfsutils-linux sudo zpool create pool1 /dev/sdX
  • TrueNAS (FreeNAS)
  • Unraid
  • 企业级归档

极简决策

选 ext4: 不知道用什么好，就用这个。系统盘最快选择。

选 XFS: 你的硬盘里有 4K 视频，或者你要跑 MySQL 数据库。

选 ZFS: 你有很多块硬盘组 NAS，或者极度害怕数据丢失（它会自动修复数据）。

关键配置命令速查

1. 格式化 (以 ext4 为例，其他把 ext4 改成 xfs/zfs 即可):

sudo mkfs.ext4 /dev/sdb1

2. 挂载:

sudo mount /dev/sdb1 /mnt/mydisk

3. ZFS 创建存储池 (进阶):

sudo zpool create -f mypool mirror /dev/sdb /dev/sdc
