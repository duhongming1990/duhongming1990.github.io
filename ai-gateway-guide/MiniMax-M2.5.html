LiteLLM - 最通用的 AI 网关

经过多维度对比（功能完备性、易用性、社区活跃度），LiteLLM 是目前开发者构建 AI 应用时，部署 AI Gateway 的首选。

查看 LiteLLM 源码

🚀 为什么选择 LiteLLM？

  • 🚀 极高的兼容性: 它是一个“Drop-in Replacement”，这意味着你不需要修改代码，只需要更改 Base URL 指向 LiteLLM 代理，它就能帮你路由到任何模型。
  • 🌐 广泛的模型支持: 支持超过 100 种模型，包括 OpenAI, Azure, Anthropic, Cohere, Ollama 等。这是目前支持提供商最多的开源网关。
  • 🛡️ 企业级特性: 内置负载均衡（支持多个 API Key 轮询）、自动重试、异常熔断、成本追踪和日志记录。
  • 🛠️ 极简部署: 只需一个简单的 Docker 命令即可启动，或者作为一个 Python 库嵌入你的项目中。

⚡ 快速开始 (Docker)

如果您想立刻部署一个 LiteLLM 代理，只需运行以下命令：

docker run \
    -d -p 4000:4000 \
    -v $(pwd)/config.yaml:/app/config.yaml \
    litellm/main:latest \
    --config /app/config.yaml --port 4000

配置文件示例 (config.yaml)：

model_list:
  - model_name: gpt-4
    litellm_params:
      model: azure/gpt-4
      api_key: os.environ/AZURE_API_KEY
      api_base: os.environ/AZURE_API_BASE
  - model_name: claude-3
    litellm_params:
      model: anthropic/claude-3-opus-200k
      api_key: os.environ/ANTHROPIC_API_KEY

litellm_settings:
  drop_params: true
  set_verbose: true
