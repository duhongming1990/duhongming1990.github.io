GCJ-02 vs WGS-84

探索两种常用地理坐标系的差异、转换及应用场景，了解它们在定位服务中的核心角色

坐标系简介

GCJ-02坐标系

GCJ-02（火星坐标系）是由中国国家测绘局制定的地理坐标系，通过对WGS-84坐标进行非线性偏移得到。

  • 中国境内强制使用的加密坐标系
  • 偏移范围约在300米以内
  • 百度地图在其基础上进一步加密为BD-09

WGS-84坐标系

WGS-84（World Geodetic System 1984）是全球通用的地球坐标系，是GPS系统使用的标准坐标系。

  • 国际通用标准坐标系
  • GPS设备直接输出的原始坐标
  • 不受地域限制，全球统一标准

坐标转换原理

WGS-84 → GCJ-02

将GPS原始坐标转换为国内地图使用的加密坐标，需要使用国家测绘局提供的偏移算法。

// 简化转换公式示例
function wgs2gcj(wgsLat, wgsLng) {
  var dLat = transformLat(wgsLng - 105.0, wgsLat - 35.0);
  var dLng = transformLng(wgsLng - 105.0, wgsLat - 35.0);
  var radLat = wgsLat / 180.0 * Math.PI;
  var magic = Math.sin(radLat);
  magic = 1 - ee * magic * magic;
  var sqrtMagic = Math.sqrt(magic);
  dLat = (dLat * 180.0) / ((a * (1 - ee)) / (magic * sqrtMagic) * Math.PI);
  dLng = (dLng * 180.0) / (a / sqrtMagic * Math.cos(radLat) * Math.PI);
  var gcjLat = wgsLat + dLat;
  var gcjLng = wgsLng + dLng;
  return [gcjLat, gcjLng];
}

GCJ-02 → WGS-84

将国内地图坐标转换为GPS原始坐标，需要使用逆向算法，通常采用迭代逼近的方式实现。

// 简化转换公式示例
function gcj2wgs(gcjLat, gcjLng) {
  var initDelta = 0.01;
  var threshold = 0.000001;
  var dLat = initDelta, dLng = initDelta;
  var mLat = gcjLat - dLat, mLng = gcjLng - dLng;
  var pLat = gcjLat + dLat, pLng = gcjLng + dLng;
  var wgsLat, wgsLng, i = 0;
  while (1) {
    wgsLat = (mLat + pLat) / 2;
    wgsLng = (mLng + pLng) / 2;
    var tmp = wgs2gcj(wgsLat, wgsLng);
    dLat = tmp[0] - gcjLat;
    dLng = tmp[1] - gcjLng;
    if ((Math.abs(dLat) < threshold) && (Math.abs(dLng) < threshold))
      break;
    if (dLat > 0) pLat = wgsLat; else mLat = wgsLat;
    if (dLng > 0) pLng = wgsLng; else mLng = wgsLng;
    if (++i > 10000) break;
  }
  return [wgsLat, wgsLng];
}

应用场景

国内地图服务

高德地图、腾讯地图、百度地图等国内主流地图服务均使用GCJ-02或其进一步加密的坐标系（如百度BD-09）。

导航与定位

国内导航软件、物流追踪系统、共享单车等服务均采用GCJ-02坐标系，确保与国内地图数据的一致性。

国际应用

Google Maps、OpenStreetMap等国际地图服务，以及GPS设备原始数据、航空航天等领域使用WGS-84坐标系。
